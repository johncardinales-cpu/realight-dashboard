Realight dashboard files

How to use:
1. Open your local project folder:
   C:\Users\Admin\realight\realight-dashboard
2. Copy these files into the matching folders.
3. Overwrite existing app/page.tsx and app/layout.tsx if prompted.
4. Then run:

   git add .
   git commit -m "Add shared layout and route files"
   git push origin main

Vercel will auto-deploy.
